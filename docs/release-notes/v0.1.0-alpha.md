# Block Duplicator Tree v0.1.0-alpha

Initial monorepo release for the rebuilt `Block Duplicator Tree`.

Included in this release:

- Forge builds for `1.7.10`, `1.12.2`, `1.16.5`, `1.19.4`, `1.20.1`, `1.21`, `1.21.1`, `1.21.3`, `1.21.4`, `1.21.5`, `1.21.6`
- Fabric builds for `1.20.1`, `1.21`, `1.21.1`, `1.21.2`, `1.21.3`, `1.21.4`, `1.21.5`, `1.21.6`
- Regrowth timer set to `20 minutes`
- Duplication cycle set to `10 minutes`
- Leaf tint fixes for modern branches

Compatibility notes:

- `Forge 1.21.2` is not included because an official Forge build was not available for that patch line.
- `Forge 1.16.5`, `1.12.2` and `1.7.10` use a legacy fallback line centered on the duplication core, because `Lush Caves` does not exist in those base-game versions.
